## Core Principles (ALWAYS Apply — Non-Negotiable)

1. **Agent = LLM + Instructions + Tools + Orchestration** — never conflate layers.
2. **Stateless handlers, stateful sessions** — code is stateless; state lives in session/Firestore.
3. **Fail loud, recover gracefully** — every tool catches errors and returns structured error dicts.
4. **One responsibility per agent** — decompose complex tasks into specialized sub-agents.
5. **Prompt is code** — version-control instructions, apply prompt engineering, test in PRs.
6. **Observe everything** — Cloud Trace + structured logging from day one.
7. **ALWAYS handle 429** — every external call MUST handle rate limiting with exponential backoff + jitter. NO EXCEPTIONS.
8. **Idempotent tools** — retrying a failed call must be safe. Design for retryability.
9. **Minimize token burn** — cap tool outputs, paginate, trim history. Tokens = cost + latency.
10. **Secure by default** — no hardcoded secrets, least-privilege SA, input validation on every tool.
11. **Follow ASP conventions** — use Agent Starter Pack project structure for production readiness.

## Agent Starter Pack (ASP) — Project Structure

> **ALWAYS follow this structure.** It is the official Google-recommended layout
> generated by `agent-starter-pack create` and compatible with ADK CLI, Agent Engine,
> Cloud Run, and GKE deployment targets.

### Create a new project

```bash
# Install and create (Agent Engine — default for conversational agents)
pip install agent-starter-pack
agent-starter-pack create my-agent -a adk -d agent_engine

# Or with uvx (no install)
uvx agent-starter-pack create my-agent -a adk -d agent_engine

# Cloud Run — for custom APIs, webhooks, external-facing agents
uvx agent-starter-pack create my-agent -a adk -d cloud_run
```

### Key ASP Conventions

| Convention | Rule |
|---|---|
| **Agent directory** | `app/` (default) or custom via `--agent-directory` |
| **Entry point** | `app/agent.py` must export `root_agent` variable |
| **`__init__.py`** | Must contain `from . import agent` |
| **Sub-agents** | Each in `app/sub_agents/{name}/` with own `agent.py` |
| **Tests** | In `tests/` with `unit/`, `integration/`, `load/` subdirs |
| **Infra** | In `deployment/terraform/` |
| **CI/CD** | In `.cloudbuild/` or `.github/workflows/` |
| **Environment** | `.env` in `app/` for local dev; Secret Manager for prod |
| **GEMINI.md** | Context file for AI assistants (Gemini CLI, Claude Code) |

from google.adk.agents import Agent

    model="gemini-3-flash",

    instruction=ROOT_INSTRUCTION,
    tools=[search_database, generate_report],
)

## Mandatory 429 Handling — Copy to Every Project

Every ADK project MUST include `app/shared/retry.py`:

```python
# app/shared/retry.py
import asyncio
import random
import time
import logging
from functools import wraps
from typing import Callable

logger = logging.getLogger(__name__)


class RetryableError(Exception):
    """Raised when an operation should be retried (429, 5xx)."""
    def __init__(self, message: str, retry_after: float | None = None):
        super().__init__(message)
        self.retry_after = retry_after


def with_rate_limit_retry(
    max_retries: int = 5,
    base_delay: float = 1.0,
    max_delay: float = 60.0,
    retryable_status: tuple[int, ...] = (429, 500, 502, 503, 504),
):
    """Exponential backoff + jitter for 429 and transient errors.

    MUST wrap every function that calls an external API, LLM, or service.
    Respects Retry-After header. Never raises — returns structured error dict.
    """
    def decorator(func: Callable):
        @wraps(func)
        async def _async(*args, **kwargs):
            last_err = None
            for attempt in range(max_retries + 1):
                try:
                    result = await func(*args, **kwargs)
                    if hasattr(result, "status_code") and result.status_code in retryable_status:
                        retry_after = getattr(result, "headers", {}).get("Retry-After")
                        raise RetryableError(
                            f"HTTP {result.status_code}",
                            retry_after=float(retry_after) if retry_after else None,
                        )
                    return result
                except RetryableError as e:
                    last_err = e
                    if attempt < max_retries:
                        delay = min(base_delay * (2 ** attempt) + random.uniform(0, 1), max_delay)
                        if e.retry_after:
                            delay = max(delay, e.retry_after)
                        logger.warning(
                            f"[RETRY] {func.__name__} attempt {attempt+1}/{max_retries} "
                            f"— {e} — waiting {delay:.1f}s"
                        )
                        await asyncio.sleep(delay)
                except Exception as e:
                    logger.exception(f"[FATAL] {func.__name__}: {e}")
                    return {"status": "error", "error": str(e), "retryable": False}
            return {
                "status": "error",
                "error": f"Rate limited — {max_retries} retries exhausted: {last_err}",
                "retryable": True,
                "suggestion": "Service is rate-limited. Wait a moment and try again.",
            }

        @wraps(func)
        def _sync(*args, **kwargs):
            last_err = None
            for attempt in range(max_retries + 1):
                try:
                    result = func(*args, **kwargs)
                    if hasattr(result, "status_code") and result.status_code in retryable_status:
                        retry_after = getattr(result, "headers", {}).get("Retry-After")
                        raise RetryableError(
                            f"HTTP {result.status_code}",
                            retry_after=float(retry_after) if retry_after else None,
                        )
                    return result
                except RetryableError as e:
                    last_err = e
                    if attempt < max_retries:
                        delay = min(base_delay * (2 ** attempt) + random.uniform(0, 1), max_delay)
                        if e.retry_after:
                            delay = max(delay, e.retry_after)
                        logger.warning(
                            f"[RETRY] {func.__name__} attempt {attempt+1}/{max_retries} "
                            f"— {e} — waiting {delay:.1f}s"
                        )
                        time.sleep(delay)
                except Exception as e:
                    logger.exception(f"[FATAL] {func.__name__}: {e}")
                    return {"status": "error", "error": str(e), "retryable": False}
            return {
                "status": "error",
                "error": f"Rate limited — {max_retries} retries exhausted: {last_err}",
                "retryable": True,
            }

        return _async if asyncio.iscoroutinefunction(func) else _sync
    return decorator
```

### Safe Tool Decorator — `app/shared/errors.py`

```python
# app/shared/errors.py
import asyncio
import logging
from functools import wraps

logger = logging.getLogger(__name__)


def safe_tool(func):
    """Catches all exceptions, returns structured error dict. Never crashes the agent."""
    @wraps(func)
    def _sync(*args, **kwargs):
        try:
            return func(*args, **kwargs)
        except Exception as e:
            logger.exception(f"Tool {func.__name__} failed")
            return {
                "status": "error",
                "error": f"{type(e).__name__}: {e}",
                "tool": func.__name__,
                "suggestion": "Try different parameters or a simpler request.",
            }

    @wraps(func)
    async def _async(*args, **kwargs):
        try:
            return await func(*args, **kwargs)
        except Exception as e:
            logger.exception(f"Tool {func.__name__} failed")
            return {
                "status": "error",
                "error": f"{type(e).__name__}: {e}",
                "tool": func.__name__,
                "suggestion": "Try different parameters or a simpler request.",
            }

    return _async if asyncio.iscoroutinefunction(func) else _sync
```

### Decorator Stack for Every External Tool

```python
from app.shared.errors import safe_tool
from app.shared.retry import with_rate_limit_retry, RetryableError

@safe_tool                              # outermost — catches everything
@with_rate_limit_retry(max_retries=5)   # middle — retries 429/5xx
async def call_api(url: str) -> dict:   # innermost — actual logic
    """Call external API.

    Args:
        url: The API endpoint URL.
    """
    async with aiohttp.ClientSession() as session:
        async with session.get(url, timeout=aiohttp.ClientTimeout(total=30)) as resp:
            if resp.status == 429:
                raise RetryableError("HTTP 429", retry_after=resp.headers.get("Retry-After"))
            if resp.status >= 500:
                raise RetryableError(f"HTTP {resp.status}")
            resp.raise_for_status()
            return {"status": "success", "data": await resp.json()}
```
